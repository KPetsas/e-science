##General Information

Prerequisites for the installation of web environment.

* Python 2.7.x
* easy_install and pip
* Git 1.7/1.8
* virtualenv
* Django
* Database (SQLite, MySQL, PostgreSQL)
* South
* Text editor (Sublime, vim, Komodo, gedit)
